# Reproducible EDDS Project (repro-edds)

This repository serves as the central location for a reproducible verification and validation (V&V) study of a simplified electronic drug delivery system (EDDS) computational model involving conjugate heat transfer.

The goal of this project is to develop credible, reproducible, and decision-ready computational modeling workflows aligned with VVUQ best practices and risk-informed assessment frameworks.

Rather than duplicating work, this repository links to modular, solver-specific implementations and analysis workflows.
